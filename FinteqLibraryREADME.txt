Getting Started

What You Need

Before diving into the Finteq Library Management System, make sure you have the following software installed on your computer:

- Visual Studio or any other development environment for C#.
- The .NET SDK to run C# applications.

Setting It Up

1. Begin by cloning the project repository to your local machine using the following command:

git clone https://github.com/yourusername/Finteq-Library-Management.git


2. Open the project in your preferred C# development environment, such as Visual Studio.

3. Build and run the application.

Application Features

1. Adding New Books

- This feature allows you to add new books to your library. You'll need to provide details like the book's title, author, and publication year.

2. Updating Book Information

- Use this feature to modify existing book details, including the title, author, or publication year.

3. Removing Books

- When a book is no longer part of your collection, you can permanently delete it from the library. Be cautious with this option.

4. Checking Out Books

- Library users can check out books for borrowing, and this action marks the book as checked out.

5. Returning Books

- When a user returns a checked-out book to the library, you can use this feature to mark the book as available again.

6. Viewing Available Books

- You can generate a list of all the books currently available in your library.

How to Use It

Adding a New Book

1. Click on the "Add Book" button on the main menu.
2. Enter the book's title, author, and publication year in the corresponding text boxes.
3. Click "Submit" to add the book to the library. A success message will confirm the addition.

Updating Book Details

1. Select the "Update Book" option on the main menu.
2. Enter the book's ID to search for the book you want to update.
3. Provide the new title, author, or publication year as needed.
4. Click "Submit" to save the changes. A success message will indicate the update.

Removing a Book

1. Choose "Delete Book" from the main menu.
2. Enter the book's ID to find the book you want to delete.
3. Confirm the deletion in the warning prompt.
4. A success message will confirm the book's removal.

Checking Out a Book

1. Click on "Check Out Book" in the main menu.
2. Enter the book's ID to locate the book you want to check out.
3. Confirm the checkout in the warning prompt.
4. A success message will confirm the book's successful checkout.

Returning a Book

1. Select "Return Book" from the main menu.
2. Enter the book's ID to locate the book you want to return.
3. Confirm the return in the warning prompt.
4. A success message will confirm the book's return.

Checking Available Books

1. Click "View Books" on the main menu.
2. A list of all available books in your library will be displayed in a data grid.

Now you have a comprehensive guide on how to use the Finteq Library Management System. Enjoy efficiently managing your library!
